# CareSync Hospital Management System

CareSync is a Java-based Hospital Management System developed using Object Oriented Programming (OOP) concepts.

## Features
- Patient Management
- Doctor Management
- Staff Management
- Appointment Scheduling
- Digital Billing System

## OOP Concepts Used
- Inheritance
- Encapsulation
- Polymorphism
- Abstraction

## Technologies
- Java
- OOP
- File Handling
